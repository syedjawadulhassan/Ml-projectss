About This Project

This web application demonstrates a credit card fraud detection system built with Python, Flask, and scikit-learn. It uses the Kaggle Credit Card Fraud Detection dataset to train machine learning models capable of distinguishing fraudulent transactions from legitimate ones in real time.

  • Data preprocessing with handling of missing values and feature scaling
  • Class imbalance treatment using SMOTE
  • Baseline and ensemble models (Logistic Regression & Random Forest)
  • Evaluation via accuracy, precision, recall, F1 score, and confusion matrix
  • Deployment with a responsive Bootstrap UI
